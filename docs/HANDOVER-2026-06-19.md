## Architecture Reference

### CSS Layer Order
```
tokens ‚Üí glass ‚Üí particles ‚Üí motion ‚Üí layout ‚Üí components ‚Üí utilities ‚Üí shaders
```
Source files: `src/00-themes.css` through `src/08-shaders.css`

### JS Modules (`window.Vitra`)
`theme` ¬∑ `particles` ¬∑ `reveal` ¬∑ `ripple` ¬∑ `modal` ¬∑ `tooltip` ¬∑ `toast` ¬∑ `dropdown` ¬∑ `spotlight`

### Theme System
`[data-theme]` on `<html>`. Valid: `light` ¬∑ `dark` ¬∑ `auto` ¬∑ `pastel` ¬∑ `neon` ¬∑ `ocean` ¬∑ `emerald`

### Token Conventions
- All CSS classes: `.vitra-*`
- All CSS custom props: `--vitra-*`
- Private JS members: `_*`
- Console warnings: `[Vitra ModuleName]` prefix, never `console.error()`

### Glass Pattern
Always follow:
```css
.component {
  background: var(--vitra-glass-bg-fallback); /* solid fallback */
}
@supports (backdrop-filter: blur(1px)) {
  .component {
    background: var(--vitra-glass-bg);
    backdrop-filter: blur(var(--vitra-blur-md));
  }
}
```
